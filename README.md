## ğŸš€ Getting Started

### Prerequisites
- A modern web browser.
- A Google Gemini API Key (for AI features).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/khdxsohee/Pawsome-Care.git
   cd Pawsome-Care
   ```

2. **Environment Setup:**
   Ensure your execution environment has access to your Gemini API key:
   ```bash
   export API_KEY='your_gemini_api_key_here'
   ```

3. **Development:**
   Open `index.html` in your favorite local development server (like VS Code Live Server) or use your preferred build tool.
